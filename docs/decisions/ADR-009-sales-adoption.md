# ADR-009: Sales Team Adoption Strategy

**Status:** OPEN
**Date:** 2026-04-10
**Category:** Operations
**Blocks:** Phase 2 (LİFFY quote integration)

---

## Question

Should LİFFY quote submission replace Zoho quote creation immediately, or run in parallel during a transition period?

## Options

| Option | Pros | Cons |
|--------|------|------|
| **Immediate replacement** | Clean cut, no confusion about which system to use | Agents may resist, bugs have no fallback |
| **Parallel for 1 expo cycle** | Agents can compare, report issues, build confidence | Dual entry again, agents might never switch |
| **Pilot with one office first** (e.g., Turkey or Nigeria) | Controlled risk, learn from real usage, then roll out | Slower, some offices on old system longer |

## Considerations

- LİFFY quote form should be simpler than Zoho (fewer fields) — this is a selling point for agents
- Agents in different countries have different tech comfort levels
- Jude's Nigeria office and Morocco office may need different timelines
